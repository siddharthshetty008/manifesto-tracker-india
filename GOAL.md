# GOAL

Ship a public, maintained, citation-backed tool that displays verified promise-fulfillment verdicts for Indian election manifestos, starting with the BJP 2014 manifesto.

Success = a live public URL + a growing count of verified promise-outcome entries + external engagement (media citations, academic references, returning visitors).

## This project is NOT

- A research paper. The paper track is a lightweight log in [PAPER_NOTES.md](PAPER_NOTES.md), never a steering goal.
- A fine-tuned model. All LLM use is API-only — Claude drafts, humans verify.
- A benchmark or evaluation harness.
- A performance-engineering portfolio.
